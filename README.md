# Delivery API

Projeto desenvolvido com Spring Boot para gerenciamento de restaurantes.

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Endpoints

### Health Check

GET /health

### Informações

GET /info

### Restaurantes

GET /restaurantes

## Executando o projeto

```bash
./mvnw spring-boot:run
